## Getting started
To build the project, you will first need to install necessary dependencies using [npm](http://npmjs.org) and [Bower](http://bower.io).

Run in the project root:

```sh
$ npm install & bower install
```

Start the server:

```sh
$ npm start
```

Or simply:

```sh
$ node app
```
